    Welcome to BRCA Lightning Help

    This site is a work in progress and is open-source. Please contact us here with questions, unexpected behavior reports, and feature requests.

    Why do some pages take a long time to load?

    This is a prototype implementation of Lightning, so some queries take longer than expected. We are working on improving implementation.

    Why do some pages freeze my browser?

    Our API do not page results, so if a user requests over 5,000 or so bases, 5,000 bases * 174 people will be loaded into the window. This amount of information transfer will freeze a browser. We suggest interacting directly with the api to retrieve that many bases.

    I can only find Chromosomes 13 and 17. Where are the others?

    We haven't loaded those chromosomes into our database yet: again, this is a prototype implementation.

    0-indexed or 1-indexed?

    Our API's support both, but it is 0-indexed by default.
