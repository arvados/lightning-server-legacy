Notes

What does this service do?

This service will search across all the genomes in the chosen population, using the given coordinate against the chosen chromosome and reference genome, for the given string of alleles. If a match is found in any of the genomes, it returns True. Otherwise, it returns False.

What are Beacons?

For more information about the motivation behind this project, as well as list of other available beacons, please see the Global Alliance for Genomics and Health's page on Beacons.
